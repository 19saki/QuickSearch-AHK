# QuickSearch-AHK 🚀

**QuickSearch-AHK** 是一款基于 **AutoHotkey v2.0** 开发的高效搜索增强工具。它将“复制、切换、粘贴、搜索”的繁琐流程压缩为一次简单的连击，旨在为开发者、研究者和 AI 重度用户提供极致的检索体验。

## ✨ 核心特性

- **双击触发 (`Double Ctrl+C`)**：在选中文本后双击复制键，立即启动“全引擎搜索”，一次性在多个预设平台打开结果。
- **智能 UI 面板 (`Ctrl+Alt+S`)**：呼出精致的半透明操作界面，支持搜索词即时二次编辑。
- **AI 深度联动**：针对 **DeepSeek**、**ChatGPT**、**Gemini** 进行了自动化适配，脚本会自动粘贴搜索内容并模拟回车，直达对话页面。
- **轻量且原生**：采用 AHK v2 编写，内存占用极低，无后台弹窗，响应极速。
- **URL 智能编码**：完美支持中文字符与特殊符号，避免搜索乱码。

## ⌨️ 快捷键说明

| 快捷键 | 动作 | 功能描述 |
| :--- | :--- | :--- |
| **`Ctrl + C` (双击)** | 静默搜索 | 自动获取选中文字，并同时开启所有预设 AI 及搜索引擎。 |
| **`Ctrl + Alt + S`** | 唤出面板 | 弹出 GUI 编辑框，可手动选择平台或修改关键词。 |
| **`数字键 1-6`** | 菜单快选 | 在面板显示时，按下对应数字即可触发特定引擎。 |
| **`Enter`** | 确认搜索 | 在面板中修改完词条后快速发动。 |
| **`Esc`** | 退出界面 | 快速关闭 UI 面板。 |

## 🛠️ 支持的引擎

1.  **综合搜索**：百度 (Baidu)
2.  **人工智能**：DeepSeek, ChatGPT, Google Gemini
3.  **内容搜索**：Bilibili (B站)
4.  **全部发动**：一键开启上述所有平台进行结果交叉对比。

## 🚀 安装与使用

### 前置要求
- Windows 10/11 操作系统
- 已安装 [AutoHotkey v2.0+](https://www.autohotkey.com/)

### 运行方式
1. 下载本仓库中的 `QuickSearch.ahk` 文件。
2. 双击脚本文件即可运行（右下角系统托盘会显示图标）。
3. 选中文本，尝试双击 `Ctrl+C` 开启你的高效搜索之旅。

---

## ⚙️ 自定义配置

你可以通过右键点击托盘图标选择 `Edit Script`，轻松修改以下内容：
- **搜索引擎 URL**：更换为你常用的垂直搜索工具（如 Google, GitHub, StackOverflow）。
- **等待时间**：根据你的网络环境修改 `Sleep` 时间，确保 AI 页面加载完成后能准确执行粘贴动作。
